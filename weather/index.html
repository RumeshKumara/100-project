Welcome to the JavaScript console!
Type console.log("Hello, World!"); to see the output.
Type console.log(1 + 2); to see the result of addition.
Type console.log(Math.random() * 100); to see a random number between 0 and 100.
Type console.log(Math.floor(Math.random() * 100) + 1); to see a random integer between 1 and 100.
Type console.log(Math.PI); to see the value of Pi.
